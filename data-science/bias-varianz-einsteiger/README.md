# Bias-Varianz Einsteiger-Spiel

Niedrigschwellige Interaktion für Teilnehmende ohne Mathe- oder Technik-Vorkenntnisse.

## Datei

- `bias_varianz_einsteiger_interaktion.html`

## Lernziel

Teilnehmende verstehen nach dem Durchlauf:

- was Bias bedeutet (systematische Schieflage)
- was Varianz bedeutet (sprunghafte Instabilität)
- warum ein ausgewogenes Modell beide Risiken reduziert
